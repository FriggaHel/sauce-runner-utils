# Sauce Runner Utils

A utilities library used by other Sauce Runners (e.g.: https://github.com/saucelabs/sauce-cypress-runner/) to share commonly used code between them

## Usage

```javascript
const sauceRunnerUtils = require('sauce-runner-utils');
sauceRunnerUtils.
```
